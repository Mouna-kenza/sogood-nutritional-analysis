## 🎯 Ma Responsabilité
Interface utilisateur complète pour l'analyse nutritionnelle

## 🛠️ Technologies
- **Flask** : Backend web Python
- **Bootstrap** : Design responsive
- **JavaScript/jQuery** : Interactivité

- **Analytics** : Graphiques et tendances
